How To Use The VHS Tuner

This wizard lets you mark bad frames, tune gamma, author people subtitle ranges, edit dialogue subtitles, and create chapter ranges. Progress saves write BAD_FRAMES + gamma metadata to metadata/<archive>/render_settings.json, people subtitle ranges to metadata/<archive>/people.tsv, dialogue subtitle rows to metadata/<archive>/subtitles.tsv, and chapter rows to metadata/<archive>/chapters.tsv. During render, subtitle files are generated from that data.

Step 1: Load Chapter

  1. Select an archive in the left column, then select a chapter.
  2. Click the 2/7 Bad Frames step button at the top to load all chapter frames. The loader can be cancelled while running.

Step 2: Bad Frames

  • Each frame card is color-coded: green is currently good, red is currently bad.
  • Click any frame to toggle its manual override between good and bad.
  • Enable Sim FreezeFrame to preview replacement in real time; dashed card borders mark simulated replacements.
  • Enable Force All Good to force every loaded frame status to good for preview/save.
  • Use the IQR k slider to change the automatic threshold. Lower k marks more frames bad; higher k marks fewer.
  • The sparkline shows score distribution and threshold. Drag on it to jump around the grid.
  • Fullscreen expands the review panel for faster triage.
  • Preview Render runs a real chapter render preview from the current in-memory bad-frame selections, without writing render_settings.json yet.

Step 3: Gamma Correction

  • The frame grid and flipbook stay the same, with FreezeFrame simulation applied.
  • The sparkline switches to gamma score mode; the horizontal bar is your selected gamma level.
  • Use Mode = Whole Chapter to apply one gamma value to the full chapter.
  • Use Mode = Visible Range Regions then click Apply Visible Range to add per-range gamma values.

Step 4: People

  • Optionally click Auto-fill From Cast for a first-pass draft from cast/data.
  • Use the people table to edit start, end, and people rows for people.tsv.
  • The people table auto-scrolls/highlights around the current timeline cursor while you scrub.
  • Leave the final blank row empty, or fill it to add a new people row.
  • Use Clear Entries to remove all people rows for the current chapter draft.
  • Use Save Progress anytime to persist people rows and resume later.

Step 5: Subtitles

  • Click Generate Subtitles to auto-transcribe the chapter into the subtitle table.
  • Use the audio waveform strip to scrub with the vertical playhead and align subtitle timing by ear.
  • Click the play button beside the waveform to audition audio from the current playhead.
  • Use the subtitle table below the timeline to edit each field cell-by-cell.
  • The subtitle table auto-scrolls/highlights to follow the timeline cursor while you scrub.
  • Use the timeline scrub slider to move quickly across chapter timecodes.
  • Each entry is written in chapter-local time format: HH:MM:SS.mmm.
  • Subtitle confidence/source metadata is preserved for saves and render, but hidden in this tuner view.

Step 6: Chapters

  • Use Chapters to define chapter sub-ranges with chapter frame numbers (inclusive end frame).
  • Chapter rows are saved to metadata/<archive>/chapters.tsv with ffmetadata columns.

Step 7: Summary + Save

  1. Click the 7/7 Summary + Save step button at the top to verify bad-frame, gamma, subtitles, and chapter settings.
  2. Click Save and Return to Chapters to persist selected bad frames, gamma ranges, people/subtitle rows, and chapter rows.
  3. The chapter list refreshes with updated BAD counts after save.

How FreezeFrame Is Applied

  • The tuner does not directly edit video pixels. It only writes frame IDs into chapter metadata.
  • In render, the AviSynth script builds FreezeFrame(start, end, source) instructions from those bad-frame ranges.
  • The source frame prefers the next clean frame; if none exists (for example at chapter end), it falls back to the nearest previous clean frame.
  • Adjacent bad-frame IDs are merged into ranges, so contiguous damage is replaced consistently.

Tips

  • If the auto threshold is too aggressive, increase IQR k before manual edits.
  • Manual overrides in the loaded frame set are saved with the chapter when you save.
